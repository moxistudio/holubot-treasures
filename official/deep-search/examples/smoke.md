# Smoke Example: Deep Search

## Input

```text
帮我研究一下 2026 年 3 月 AI Agent 开发框架的主要发布动态。给我 3 条关键结论，并附来源。
```

## Expected Lane

`medium`（Nimbus research chain）

## Expected Output Style

- 先给一句话结论
- 然后给 3-6 条关键发现
- 附 1-5 条关键来源（标题或链接）
- 有不确定项时单列“待验证”
- 不输出原始 JSON，不输出内部调度细节

## Non-Goals

- 不保证覆盖全网全部信息
- 不在无来源时伪造来源
- 不承诺替代正式尽调报告
