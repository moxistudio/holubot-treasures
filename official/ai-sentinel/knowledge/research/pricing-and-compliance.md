# Pricing And Compliance

## Pricing logic

The DR-4 synthesis converged on a practical China-market subscription range around:

- RMB 19.9 per month for a basic tier
- RMB 39.9 per month for a professional tier

The real point is not the exact number. It is that the user should feel a clear exchange:

- less noise
- faster understanding
- clearer decisions
- cumulative memory

## Value model

The paid wall should protect the multiplication of:

- information quality
- structural usefulness
- depth of insight

If any one of those collapses, the product starts to look like a commodity feed.

## Packaging logic

The synthesis suggested a useful split:

- free: sparse or weekly selected brief
- basic paid: daily full brief plus short history
- professional paid: longer history, trend reports, stronger query features, more personalization

## Compliance logic

The China-side synthesis matters because compliance is not a side topic here. It affects:

- source legality
- attribution and traceability
- claim safety
- operational responsibility

## Product implication

For AI Sentinel, compliance itself can be part of the value proposition:

- lawful sources
- traceable evidence
- explicit confidence
- explicit pending-verification markers
- no careless rumor amplification

## Working rule

When in doubt:

- prefer attributable sources
- prefer explicit uncertainty over overstated certainty
- prefer safe public claims with richer internal notes kept separate
